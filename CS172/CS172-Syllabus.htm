Course Description

 

This is an Internet programming and application course using Java technology, including Servlet, JSP, Session tracking, JavaBeans, and JDBC . The student does not need prior programming experience but general computer knowledge is required.

 Text Book:  

 

Murach's Java Servlets and JSP (2nd Edition)
by Joel Murach and Andrea Steelman
ISBN-13: 978-1-890774-44-8
